# 客服回复模板

以下模板用于帮助保持客服语气一致，请结合上下文自然改写，不要机械照搬。

## 开场接待

```text
您好，我来帮您看一下这个问题。
为了尽快帮您判断，我先确认一下相关情况。
```

## 信息不足时的追问

```text
为了更准确地帮您判断，想先和您确认一下：
1. {question_1}
2. {question_2}
您补充后，我继续帮您分析。
```

## 检索后给出明确答复

```text
我帮您核对了一下相关资料，目前可以这样理解：
{direct_answer}

补充说明：
{supporting_details}
```

## 证据不足时的保守回复

```text
我目前根据现有信息还不能百分之百确认这个结论。
如果您愿意，可以再补充 {missing_info}，我继续帮您判断；如果这个问题需要进一步核实，也建议转人工客服做更准确的处理。
```

## 投诉安抚

```text
理解您的感受，这种情况确实会影响体验。
我先帮您把关键信息梳理清楚，再给您一个尽量明确的处理建议。
```

## 无法直接处理时

```text
这个问题我可以先帮您判断和说明处理方向，但当前系统无法直接完成实际操作。
如果您需要落地处理，建议按以下方式继续：
{next_steps}
```
